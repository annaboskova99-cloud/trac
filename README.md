# Trucking Bot 🚛
**python-telegram-bot v21 + SQLite**

## Архитектура

```
1 бот → много групп
1 группа = 1 водитель + бот + оператор
Роли: оператор (полный доступ) / водитель (только получает)
```

## Функционал

| Раздел | Возможности |
|---|---|
| 👥 Водители | Добавить, редактировать, активировать/деактивировать, удалить |
| 📋 Шаблоны | PTI, DOT, безопасность, кастомные — создать/редактировать/удалить/отправить |
| 🕐 Расписания | Гибкий конструктор: каждый день, дни недели, интервал, первая неделя месяца |
| 📨 Рассылка | Текст / фото / файл → одному, нескольким или всем водителям |

## Быстрый старт

### 1. Установка

```bash
pip install -r requirements.txt
```

### 2. Настройка `config.py`

```python
BOT_TOKEN    = "токен от @BotFather"
OPERATOR_IDS = [ваш_telegram_id]   # узнать: написать @userinfobot
```

Или через переменные окружения:

```bash
export BOT_TOKEN="ваш_токен"
export OPERATOR_IDS="123456789"
```

### 3. Запуск

```bash
python bot.py
```

## Добавление водителя (инструкция)

1. Создайте группу в Telegram
2. Добавьте бота и водителя в группу
3. Напишите @userinfobot в группу → скопируйте ID чата
4. В панели оператора: **👥 Водители → ➕ Добавить**
5. Введите имя и ID группы

## Форматы расписаний

| Формат | Пример | Описание |
|---|---|---|
| `HH:MM` | `09:00` | Каждый день |
| `HH:MM\|дни` | `08:00\|mon,wed,fri` | Пн, Ср, Пт |
| `HH:MM\|1` | `09:00\|1` | Первая неделя месяца |
| `*/Nh` | `*/4h` | Каждые N часов |
